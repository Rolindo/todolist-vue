## 2. 说明


### 1. 使用Vue脚手架vue-cli搭建项目

安装：

```
npm install -g vue-cli
```

使用：

```
vue init webpack 
```

### 2. 使用stylus 

安装：

```
npm install -D stylus stylus-loader
```


### 3. 下载运行
1. 安装依赖
```
npm install 
```
2. 运行
```
npm dev
```
3. 打包
```
npm build
```
